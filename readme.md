# Z-Tech

## Tecnologias

- Node.js
- TypeScript
- PostgreSQL
- TypeORM
- Jest
- Docker & Docker compose
- ESLint

Integrado com GitHub Actions

## Executando

Exporte as funções shell

```bash
source dev.sh
```

Inicie o setup

```bash
setup_dev_environment
```

Inicie a aplicação em dettach mode

```bash
dkupd
```

Rode os testes

```bash
dktest
```
